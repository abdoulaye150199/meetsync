Place your Jersey 10 font files here with the following names (examples):

- Jersey10-Regular.woff2
- Jersey10-Regular.woff
- Jersey10-Bold.woff2  (optional)
- Jersey10-Bold.woff   (optional)

If you don't have the font files, obtain them from your license/source and drop them into this folder. The project expects these filenames; if you use other names, update `src/index.css` accordingly.

After adding the files, run:

```
npm run dev
```

to verify the site loads the local font.
